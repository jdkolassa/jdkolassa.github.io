You know, we&#8217;ve become too fixated on the internet. It&#8217;s gotten to a point where, if we have a computer that is unable to connect, said computer is useless.

Want to edit a video using Adobe? Sorry, you need the internet to verify your program.

Want to see if you can upgrade to Windows 7? Sorry, you need to connect to Microsoft&#8217;s server to check for compatibility issues.

Want to play games? Nope, need the Internet for that too. Even in &#8220;offline&#8221; mode.

Look, I get the Internet is amazing. I love it myself. But when you&#8217;re trying to fix an old computer, or just _use_ the damn thing, and it doesn&#8217;t have working Internet, it makes it very difficult because all the programs out there are _required_ to use it. This is rather silly. Internet access should not be a base _requirement_ for a program, it should be an added benefit.

But nobody thinks that way and you run into absurd problems like the one I have, which is I&#8217;m trying to fix my computer, but am completely unable to do so. I don&#8217;t even know if I can upgrade to Windows 7 at this point. What a bummer.
